# Cleft SDK

Framework-agnostic agent storage for Filecoin Onchain Cloud (FOC).

Cleft gives AI agents durable, restart-safe memory with a verifiable audit trail.

- Checkpoint chain for resumable state
- Append-only logs for audits
- Wallet-aware storage operations
- Portable ERC-8004 identity

See [getting-started](../getting-started.md) and [modules](./modules.md).
