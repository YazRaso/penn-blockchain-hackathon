# Cleft

Cleft is a framework-agnostic agent storage SDK for Filecoin Onchain Cloud.

It gives autonomous agents durable memory and production-safe continuity through:

- checkpointed, resumable state on Filecoin,
- append-only audit logs for verifiable history,
- ERC-8004 onchain identity for portable agent continuity,
- wallet-aware operation for autonomous storage behavior.

In short, Cleft helps agents survive restarts, prove what happened, and keep operating without manual babysitting.
